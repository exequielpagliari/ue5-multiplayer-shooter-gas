# Protocol: Rift Arena

Protocol: Rift Arena is a learning-focused Unreal Engine multiplayer shooter project built with C++, Gameplay Ability System, Enhanced Input, Gameplay Tags, replication, and a modular architecture inspired by Lyra.

The goal is not to build a commercial game, but to study and document modern gameplay systems through a progressive, portfolio-oriented project.

## Core Idea

A small third-person multiplayer arena shooter where two teams fight to control experimental energy zones using weapons, roles, and GAS-driven abilities.

## Project Goals

- Learn Unreal Engine C++ for gameplay systems.
- Understand Unreal's Gameplay Framework.
- Learn multiplayer replication and server-authoritative gameplay.
- Build abilities using Gameplay Ability System.
- Use Enhanced Input and Gameplay Tags.
- Create a basic shooter weapon system.
- Implement teams, score, death and respawn.
- Build a simple objective-based game mode.
- Explore a simplified Lyra-inspired architecture.
- Document every major system for portfolio purposes.

## What This Project Is Not

This is not a commercial game.
This is not a live-service product.
This is not a full competitive shooter.
This is not an attempt to clone Lyra.

This is a structured learning project.
